### map() 함수
- map(f, iterable)은 함수(f)와 반복 가능한 데이터를 입력 받는 함수다. 
- 즉, 각 요소에 함수(f)값을 적용시킨다.
### split() 함수
- 문자열 관련 함수로, 문자열을 나눈다.
- 괄호 안에 아무것도 넣어주지 않으면 공백을 기준으로 문자열을 나눈다.
- 괄호 안에 특정 값을 넣어줄 경우 그 값을 기준으로 문자열을 나눈다.

``` 
a = "hi python"
a.split()
['hi', 'python']

b = "a,b,c,d"
b.split(',')
['a', 'b', 'c', 'd']
```
